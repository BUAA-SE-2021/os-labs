# RAID 阵列初体验

## 实验目的

通过RAID原理实践理解阵列相关知识

## 实验内容

本实验通过软件RAID管理磁盘：

- 管理工具-计算机管理-存储-磁盘管理
- C盘扩展卷
- 检查其它磁盘是否联机,联机并初始化磁盘
  - 尝试创建跨区卷并存储部分数据
  - 尝试创建镜像卷并存储部分数据
  - 尝试创建RAID5卷并存储部分数据
- 通过设备管理禁用其中一块磁盘,模拟磁盘损坏.观察有什么情况发生,数据是否损坏或者丢失.
